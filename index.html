Git Commands Cheatsheet

Commands

  • git clone
  • git status
  • git add
  • git commit -m
  • git push

This website is helpful.

git clone

With git clone I can clone a git repository into my computer's folder structure.
Example: git clone https://github.com/JessNap/git-cheatsheet.git

git status

With git status I can see the current status of my code. It will show unsaved files in red and saved files in green.

git add

With git add I can add files to the staging area. If I add a fullstop "." at the end everything in the folder will be added.

git commit

With git commit -m I will create a snapshot of the changes I have made and save it to the git directory.
Example: git commit -m "Message to go with the commit here"

git push

With git push I send local commits to the repository on git./p>
